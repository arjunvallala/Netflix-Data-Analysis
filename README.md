# 🎬 Netflix Data Analysis using Python

An Exploratory Data Analysis (EDA) project on the Netflix Movies and TV Shows dataset using **Python, Pandas, Matplotlib, and Seaborn**. The project focuses on understanding Netflix's content library by analyzing trends, genres, ratings, countries, release patterns, durations, and more.

---

## 📌 Project Objectives

- Perform data cleaning and preprocessing.
- Explore the distribution of Movies and TV Shows.
- Analyze Netflix content growth over the years.
- Identify the most common genres, ratings, directors, and actors.
- Study country-wise content distribution.
- Analyze content addition patterns by month and day.
- Generate meaningful business insights from the dataset.

## 📈 Key Insights

- Around **70%** of Netflix's catalog consists of **Movies**, while **30%** are **TV Shows**.
- Netflix's content library experienced rapid growth after **2015**.
- **TV-MA** is the most common content rating.
- The average movie duration is approximately **100 minutes**, while TV Shows average **1.7 seasons**.
- **United States** contributes the largest amount of Netflix content, followed by **India**, **United Kingdom**, and **Canada**.
- **International Movies** and **Dramas** are the most common genres.
- **Rajiv Chilaka** has directed the highest number of titles in the dataset.
- **Anupam Kher** appears most frequently among actors.
- Netflix adds the highest number of titles in **July**, with a large proportion added on the **1st day of each month**.

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Netflix-Data-Analysis.git
```

2. Navigate to the project folder

```bash
cd Netflix-Data-Analysis
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open **Netflix_Data_Analysis.ipynb** and run all cells.
